## To Do List API

Projeto de _To Do List_ para exemplos de implementações.<br>

Para executar a API:
```bash
dotnet run --project src/Lab.Todo.Api
```

Para executar com `watch`:
```bash
dotnet watch run --project src/Lab.Todo.Api
```
